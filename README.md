# Dashboard Bobinas BSW

Dashboard de controle de matéria-prima de bobinas BSW que lê dados diretamente do SharePoint.

## Como funciona

O app se conecta ao SharePoint via Microsoft Graph API, baixa o arquivo Excel mais recente e exibe o dashboard atualizado automaticamente.

## Deploy no Streamlit Cloud

1. Suba este repositório no GitHub
2. Acesse https://share.streamlit.io
3. Conecte o repositório
4. Configure os secrets no painel do Streamlit Cloud
